# Core Ecom Project

Одностраничный интернет-магазин на React + TypeScript + Vite с каталогом товаров, фильтрацией по категориям и корзиной.

## Стек
- React 19, TypeScript, Vite
- React Router DOM 7
- Redux Toolkit + React Redux
- Tailwind CSS 4
- Icons: lucide-react

## Возможности
- Каталог товаров с фильтрацией по категориям и синхронизацией с URL (`/products?category=...`).
- Добавление/удаление товаров из корзины, изменение количества, расчёт итогов.
- Главная страница с промо-блоком, категориями, подборкой новинок и подпиской на рассылку.
- Адаптивная верстка (мобильное меню, сетки карточек и формы).

## Быстрый старт
```bash
# установка зависимостей
npm install

# если react-router-dom не подтянулся автоматически
npm install react-router-dom@7.13.1

# запуск дев-сервера
npm run dev
```
Откройте URL из консоли (по умолчанию http://localhost:5173).

## Скрипты
- `npm run dev` — запуск Vite в режиме разработки.
- `npm run build` — типизация и сборка продакшн-бандла.
- `npm run preview` — предпросмотр собранного приложения.
- `npm run lint` — проверка ESLint.

## Структура
- `src/pages` — страницы (Home, ProductCatalog, ShoppingCart).
- `src/components` — UI-компоненты и блоки (Navbar, Footer, карточки, формы).
- `src/slices` — Redux-slices для товаров и корзины.
- `src/store` — конфигурация Redux store.
- `public` — статика (изображения категорий и пр.).

## Маршруты
- `/` — главная.
- `/products?category=electronics` — каталог с выбранной категорией (поддерживаются electronics, jewelery, men's clothing, women's clothing, all).
- `/cart` — корзина.

## Известные нюансы
- Скрипт установки `npm install` может потребовать разрешений на выполнение PowerShell-скриптов — при необходимости запустите терминал с соответствующими правами или выполните команду через `npm.cmd`.
